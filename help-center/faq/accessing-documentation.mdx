## Overview

Paperguide-pay organizes all your documentation by project.

## Steps

1. Go to `https://docs.paperguide-pay.com`
2. Log in with your account
3. Select your project from the dashboard
4. Click the "Documentation" tab

You'll see all your docs for the project in one place.

<Callout kind="tip">
Bookmark the docs page for quick access later.
</Callout>
