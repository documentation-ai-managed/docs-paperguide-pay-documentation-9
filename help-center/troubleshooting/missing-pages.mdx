---
title: Documentation Pages Not Showing
description: What to do if project docs are missing.
---

## Symptoms

- Documentation tab appears empty
- Missing pages for your project

## Possible Causes

- No docs have been created
- Access permissions issue

## Quick Fix

<Steps>
  <Step title="Check project selection">
    Make sure the correct project is chosen on the dashboard.
  </Step>
  <Step title="Verify permissions">
    Confirm your account has viewer or editor access for the project.
  </Step>
  <Step title="Create a page">
    Try adding a new page. If successful, older docs may not exist yet.
  </Step>
</Steps>
