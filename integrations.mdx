## Webhook Setup for Notifications

Set up webhooks to receive events from Paperguide-pay, such as document updates or publish events. Webhooks enable real-time automation with tools like Slack or Zapier.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to your dashboard settings. Go to `<strong>Integrations > Webhooks</strong>` and click `<kbd>New Webhook</kbd>`.
  </Step>
  <Step title="Configure Endpoint" icon="link">
    Enter your endpoint URL, such as `https://your-webhook-url.com/paperguide-pay`. Select events like `doc.updated` or `doc.published`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test event and verify receipt. Save the webhook to activate it.
  </Step>
</Steps>

Here is an example payload for a `doc.updated` event:

<CodeGroup tabs="JSON">
  ```json
  {
    "event": "doc.updated",
    "doc_id": "pgpay_123456",
    "timestamp": "2024-10-15T10:30:00Z",
    "changes": {
      "title": "Updated Guide",
      "section": "integrations"
    }
  }
  ```
</CodeGroup>

## API Access for Custom Connections

Use the Paperguide-pay REST API to build custom integrations. Authenticate with your API key and interact with docs programmatically.

<Tabs>
  <Tab title="JavaScript" icon="code">
    ```javascript
    const response = await fetch('https://api.example.com/v1/docs', {
      headers: {
        'Authorization': `Bearer ${YOUR_API_KEY}`,
        'Content-Type': 'application/json'
      }
    });
    const docs = await response.json();
    console.log(docs);
    ```
  </Tab>
  <Tab title="Python" icon="python">
    ```python
    import requests
    headers = {
        'Authorization': f'Bearer {YOUR_API_KEY}',
        'Content-Type': 'application/json'
    }
    response = requests.get('https://api.example.com/v1/docs', headers=headers)
    docs = response.json()
    print(docs)
    ```
  </Tab>
</Tabs>

<ParamField path="doc_id" param-type="string" required="true">
  Unique document identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

## Embedding External Content

Embed third-party content like Figma designs or Google Docs directly into your Paperguide-pay pages.

<Expandable title="Embed Steps" default-open="true">
  1. Copy the embed code from the external service.
  2. In the Paperguide-pay editor, switch to HTML mode and paste the `<iframe>` snippet.
  3. Adjust width and height for responsiveness.

  Example embed:
  ```html
  <iframe src="https://www.figma.com/embed?embed_host=share&url=https://www.figma.com/file/abc123" width="100%" height="600"></iframe>
  ```
</Expandable>

## Syncing with Project Management Tools

Connect Paperguide-pay to Jira, Asana, or Trello to keep documentation in sync with project tasks.

<Callout kind="info">
  Syncing requires OAuth setup. Visit `<a href="https://dashboard.example.com/integrations/project-tools">Project Tools Integrations</a>` for guided setup.
</Callout>

| Tool   | Sync Features                  | Setup Time |
|--------|--------------------------------|------------|
| Jira   | Tasks → Docs, Comments sync   | 5 minutes |
| Asana  | Milestones → Publish triggers | 3 minutes |
| Trello | Cards → Doc sections          | 2 minutes |

Use webhooks or the API to trigger syncs on task updates. For advanced setups, refer to the [API documentation](/authentication).

This setup enhances your workflow by automating documentation across your stack. Start with GitHub and Slack for immediate value, then expand to custom APIs.
